# ИИ-ассистент для автосервиса «Юз»

> **Проект:** Интеллектуальный виртуальный помощник для автоматизации коммуникации с клиентами автосервиса  
> **Статус:** ✅ Реализован  
> **Технологии:** Qwen, Suvvy.ai, Google Calendar API

---

## 📝 Описание проекта

**«ИИ-ассистент для автосервиса Юз»** — это интеллектуальный чат-бот, созданный на базе современных языковых моделей для автоматизации первичного взаимодействия с клиентами. Ассистент способен вести осмысленный диалог, отвечать на частые вопросы по услугам и ценам, а также самостоятельно записывать клиентов на обслуживание с синхронизацией в Google Календарь.

Проект направлен на повышение качества обслуживания, снижение нагрузки на администраторов и минимизацию потерь заявок в нерабочее время.

---

## 🎯 Цели и задачи

### Цели
- ✅ Автоматизировать до 80% типовых запросов клиентов
- ✅ Обеспечить круглосуточную поддержку без увеличения штата
- ✅ Сократить время обработки заявки на запись до <1 минуты
- ✅ Повысить конверсию из обращения в запись

### Задачи
- [x] Разработка системного промпта для точного позиционирования бренда
- [x] Структурирование базы знаний автосервиса
- [x] Интеграция с Google Calendar для управления расписанием
- [x] Настройка NLU для распознавания намерений клиента
- [x] Тестирование диалоговых сценариев и отладка edge-кейсов

---

## ⚙️ Функциональные возможности

### 🗣️ Осмысленное ведение диалога
- Понимание контекста и сохранение истории общения в рамках сессии
- Распознавание намерений: `запись`, `вопрос_об_услуге`, `цена`, `контакты`, `жалоба`
- Адаптивный стиль общения: от формального до дружелюбного
- Обработка уточняющих вопросов и многошаговых сценариев

---

## 🛠️ Используемые сервисы

### 🔹 Qwen (от Alibaba Cloud)
> **Роль в проекте:** Языковая модель для генерации ответов и работы с базой знаний

- ✅ Создание детального системного промпта с учётом специфики автосервиса
- ✅ Подготовка и структурирование базы знаний (документы, прайсы, инструкции)
- ✅ Поддержка RAG-подхода: поиск релевантных фрагментов перед генерацией ответа
- ✅ Настройка параметров генерации: температура, top_p, max_tokens для баланса креативности/точности

🔗 [Документация Qwen](https://help.aliyun.com/zh/qwen)

### 🔹 Suvvy.ai
> **Роль в проекте:** Платформа для сборки, обучения и деплоя ИИ-ассистента

- 🧩 Визуальный конструктор диалоговых сценариев
- 📊 Панель аналитики: метрики диалогов, конверсия, ошибки распознавания
- 🔌 Готовые интеграции: Google Calendar, Telegram, WhatsApp, веб-виджеты
- 🛡️ Безопасность: шифрование данных, соответствие 152-ФЗ
- 🔄 A/B-тестирование промптов и сценариев

🔗 [Suvvy.ai](https://suvvy.ai/)
